## Run Locally

Clone the project

```bash
  https://github.com/ajaychaurasiya-cs/tv-show.git
```

Go to the project directory

```bash
  cd tv-show
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm start
```
